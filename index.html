Лобби систем

Таны тоглоомын туршлагыг өргөжүүлэх олон боломж бүхий лобби систем. Дүрээ өөрчил, хөрөнгөө удирдаж, шилдгүүдийн жагсаалтад өрсөлдөөрэй.

Агуулах

NFT интеграцтай өвөрмөц Trail болон Skin цуглуулаарай. Тоглоомын хөрөнгөө өмчлөн, чөлөөтэй арилжаалах боломжтой.

Дүр тохируулга

Өвөрмөц skin болон trail-ээр баатраа чимэглээрэй. Өөрийн онцлог төрхөөр бусдаас ялгар.

Түрийвч систем

Ард-Койнтоор тоглоомд нэвтрэх. 50 Ард-ын тогтмол орох төлбөр нь өрсөлдөөний шагналын сан үүсгэнэ.

Шилдгүүдийн самбар

Дэлхий даяар өрсөлдөөрэй! Цуглуулсан тугаар эрэмбэ тодорхойлогдоно. Өндөр байршилд шагнал их.

Тусгай арга хэмжээ

Тоглогчдыг татсан тусгай үйл явдлууд. Онцгой сорилт, хязгаарлагдмал шагнал, олон нийтийн өрсөлдөөн.

Ард-Койн эдийн засаг

Тогтмол 50 Ард орох төлбөр нь тоглогчдын сан үүсгэнэ. Шагналыг эрэмбийн дагуу хуваарилна.

Ард-Койн эдийн засаг

Алхам 1: Орох төлбөр

Тоглоом бүрд 50 Ард-ын тогтмол төлбөр. Бүх тоглогчдод ойлгомжтой, урьдчилан тооцоолох боломжтой зардлын бүтэц.

01

Алхам 2: Шагналын сан

Бүх орох төлбөр шагналын санд хуримтлагдана. Олон тоглогч өрсөлдөх тусам шагналын сан томорч, урамшуулал нэмэгдэнэ.

02

Алхам 3: Эрэмбийн систем

Тоглогчдыг цуглуулсан тугаар эрэмбэлнэ. Өндөр эрэмбэтэй тоглогчид шагналын сангаас илүү их хувь авна.

03

Алхам 4: Шагнал хуваарилалт

Шагналыг эрэмбийн дагуу өөр өөр жингээр хуваарилна. Шилдэг тоглогчид хамгийн их шагнал авах нь өрсөлдөөнийг дэмжинэ.

Технологийн бүтэц

Орчин үеийн технологиор бүтээгдсэн

Хиймэл оюун ухаан нэмэх боломжтой, API-тай бүрэн нийцтэй архитектур. Grafana ашиглан бодит цагийн хяналт, асуудал илрүүлэх, эрсдэл удирдах боломжтой.

Арын систем

Өндөр гүйцэтгэлтэй API эндпойнтуудад зориулсан FastAPI + Python. Олон тоглогчдод зориулсан өргөтгөх боломжтой архитектур.

Урд талын систем

Хурдан ачаалагддаг тоглоомд зориулсан Vanilla JS, CSS, HTML. Хүнд framework-гүй, цэвэр гүйцэтгэл.

Хиймэл оюун ухаантай

API нийцтэй загвар нь хиймэл оюун ухааны технологийг хялбар нэгтгэх боломж олгоно. Machine learning-д бэлэн.

Хяналт шинжилгээ

Тоглоомын бодит цагийн хяналтад зориулсан Grafana самбар. Асуудал илрүүлэх, гүйцэтгэлийг сайжруулах.
